## How to Capture Screenshots

1. Visit [Sepolia Etherscan](https://sepolia.etherscan.io/)
2. Search for the contract address: 0xa0eE56B7697846877d0E90FA654377dcDD68Aaa8
3. Navigate to the "Transactions" tab
4. Click on individual transactions
5. Capture full-page screenshots showing all relevant details
6. Save with the appropriate filename in this directory

## Screenshot Requirements

- **Format**: PNG or JPG
- **Resolution**: At least 1920x1080 (Full HD)
- **Content**: Full transaction details visible
- **Privacy**: Ensure no private keys or sensitive wallet information is shown
